# The Agent-Native Enterprise

**A reference operating model for organizations where any role can be filled by a human or an agent, every process is built for agents first, and every flow can be paused, adjusted by a human, and resumed.**

The Agent-Native Enterprise is an operating model for organizations built for AI agents first, keeping humans only where judgment and accountability are irreplaceable.
A role is a contract, not a person: an agent, a human, or a plain program can fill the same seat interchangeably.
Every flow carries a handbrake — any authorized human can pause it, inspect what happened and why, inject a correction, and resume from that exact point.
Humans sit above the loop as a Board — a hundred people or a single founder — writing a constitution that is compiled into machine-readable rules checked before every action.
Agents never author their own rules, and a human stepping into a seat inherits the seat's authority, never their title's.
Four operating roles run the value chain — Direction, Orchestration, Execution, Verification —
and three system roles keep the machine honest: a Steward that repairs, an Optimizer that matches capability to task, and an Auditor that rates versions and can stop a dangerous one, but never restart it — un-pausing is a human act.
Risk is graduated by blast radius: reversible routine work runs fully autonomous; irreversible outward-facing acts require a human.
Autonomy is earned from evidence and ratified by humans, never self-granted.
The unit is a sovereign cell — a whole company, or one team inside an enterprise; cells federate through treaties between Boards, never by reaching inside one another.
The model names what it cannot prevent — a corrupted Board, a compromised store, prompt injection — and contains those risks rather than denying them.
In one line: humans hold the offices and write the law; agents fill the roles and run the work; anything can be stopped, corrected by hand, and resumed — under a law that binds everyone.
